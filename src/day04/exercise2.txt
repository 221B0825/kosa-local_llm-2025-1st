1. 한국어가 지원되는 작은 모델을 하나 선택하고 프롬프트를 테스트한 후 커밋하세요. 어떤 모델을 사용해도 되며, 어떤 방법으로 검증해도 좋으나 한국어가 어느 정도 잘 지원된다는 것이 분명하게 결과에 드러나야 합니다. 학습에 사용할 모델이므로 huggingface 포맷 모델을 얻을 수 있어야 합니다.
2. 영-한 또는 한-영 데이터를 확보하여 학습하세요. 영문을 입력하면 한글을, 한글을 입력하면 영문을 출력하는데, 어느 한쪽만 번역할 수 있으면 됩니다. 영-한 또는 한-영 페어 데이터셋이 있다면 그대로 사용하면 되고, 없다면 변환하세요. 데이터를 변환했다면 huggingface repo에 push하세요.
3. 모델을 학습하는 동안 streamlit으로 한-영 또는 영-한을 번역해주는 번역용 인터페이스를 구축하세요. 로컬 LLM으로 연동하며 어떤 방법을 사용해도 되고, 어떤 모델을 사용해도 좋으나 학습한 모델로 대체할 것을 생각하여 개발하세요.
4. 학습이 끝난 모델을 huggingface repo에 저장하고, 모델이 한-영 또는 영-한 번역을 잘 한다는 것을 확인하세요. 검증된 결과를 커밋하세요. 검증된 모델을 streamlit 어플리케이션에 연동하세요.
